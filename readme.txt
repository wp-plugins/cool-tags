== Description ==

A plugin for WordPress, You can control the tag cloud(include color,font size,number,and so on) of sidebar.
	
== Installation ==

1. Extract and upload `cool-tags` to the `/wp-content/plugins/` directory  
2. Activate the plugin through the 'Plugins' menu in WordPress  
3. It works.
== Frequently Asked Questions ==

= how to use? =  

Activate the plugin through the 'Plugins' menu in WordPress,and set different color of the tag cloud,also,you can activate the widget of this plugin to control the font size,number of the tag cloud.
